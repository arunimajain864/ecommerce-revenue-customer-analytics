# E-Commerce Revenue & Customer Analytics (SQL + Power BI)

## Project Overview
This project analyzes e-commerce sales and customer behavior data using SQL, Excel, and Power BI to generate actionable business insights.

The project focuses on:
- Revenue analysis
- Customer analytics
- Product performance
- Retention metrics
- Trend analysis
- Business performance optimization

---

## Tools Used
- MySQL
- Excel
- Power BI

---

## Key Analyses Performed

### Revenue Analytics
- Monthly Revenue Trends
- Revenue Growth Analysis
- Revenue by State

### Customer Analytics
- High-Value Customers
- Customer Lifetime Value (CLV)
- Repeat Purchase Rate
- Cohort Analysis

### Product Analytics
- Product Performance
- Orders by Product Category
- Product Revenue Analysis

---

## SQL Concepts Used
- JOINs

- Aggregate Functions
- Date Functions
